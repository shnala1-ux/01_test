# CLAUDE.md

이 파일은 이 저장소에서 작업하는 Claude Code(claude.ai/code)에게 안내를 제공합니다.

## 프로젝트 개요

빌드 단계, 프레임워크, 패키지 매니저 없이 동작하는 정적 HTML 페이지 모음입니다. 각 페이지는 HTML + CSS + JS가 하나의 파일에 포함된 독립 구조입니다.

## 파일 목록

- `index.html` — 이력서 웹페이지 (HTML/CSS)
- `quotes.html` — 명언 뷰어 (HTML/CSS/JS, 다크 테마, 페이드 애니메이션)

## 개발 방법

`.html` 파일을 브라우저에서 직접 열면 됩니다. 별도 서버 불필요.

라이브 리로드가 필요한 경우 (선택):
```
npx live-server .
```

## 코딩 규칙

- 스타일은 각 파일의 `<style>` 태그에 내장 — 파일이 커지지 않는 한 외부 CSS 파일 사용 금지.
- JavaScript는 바닐라만 사용 — 라이브러리 및 프레임워크 사용 금지.
- `quotes.html`의 색상 체계는 다크 배경(`#0f0f1a`) + 골드 포인트(`#e8b86d`) — 새 페이지 추가 시 일관성 유지 또는 의도적으로 구분.
- UI 텍스트는 한국어, 코드 식별자(클래스명, 변수명 등)는 영어 사용.
